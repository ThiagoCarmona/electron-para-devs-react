# Contribuindo

Este é um repositório de aprendizado. Contribuições são bem-vindas!

## Como contribuir

1. Abra uma issue descrevendo o problema ou sugestão
2. Fork o repositório
3. Crie uma branch para sua contribuição
4. Faça um pull request

## Tipos de contribuição

- Correção de erros no conteúdo ou código
- Melhorias nas explicações
- Tradução de termos técnicos
- Sugestões de exercícios extras

## Padrões

- Mantenha o tom conversacional e acessível
- Use analogias com o mundo React web sempre que possível
- Código deve ser simples e didático (não otimizado demais)
- Cada lição deve ser independente o suficiente para fazer sentido sozinha
